# Telegram Business Game Bot 🎲🏦

A fully-featured, multiplayer Telegram bot to play the classic "International Business" (Monopoly) board game directly in your group chats!

## Features 🌟
- **Dynamic Colored UI**: Powered by Kurigram, the bot uses randomly generated colored inline buttons with custom premium emojis for an aesthetic "colorbotron" look!
- **Stylish Fonts**: Custom English fonts to give a premium feel to menus and game announcements.
- **Multiplayer Lobbies**: Play with friends in group chats.
- **Full Economy System**: Buy properties, upgrade to hotels, collect rent, and draw Chance/UNO cards.

## Local Setup 🛠️
1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Rename `.env.sample` to `.env` and fill your variables.
4. Run: `python3 -m BUSINESS`

## Requirements
- Python 3.9+
- MongoDB Database
