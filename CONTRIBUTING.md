# Contributing to Business Game Bot

Thank you for your interest in contributing to **Business Game Bot**! We welcome contributions from the community to help make this Telegram bot better.

## How to Contribute
1. **Fork the Repository:** Create a personal fork of the project on Github.
2. **Clone the Fork:** Clone the fork to your local machine.
3. **Create a Branch:** Create a feature branch (`git checkout -b feature/AmazingFeature`).
4. **Make Changes:** Add your improvements, new properties, or chance cards.
5. **Commit:** Commit your changes (`git commit -m 'Add some AmazingFeature'`).
6. **Push:** Push to the branch (`git push origin feature/AmazingFeature`).
7. **Pull Request:** Open a Pull Request from your fork to the main repository.

## Reporting Bugs
If you find a bug, please use the GitHub Issue Tracker to report it. Provide as much detail as possible, including steps to reproduce.

## Feature Requests
We'd love to hear your ideas! Open an issue using the "Feature request" template.

## Code Style
- Follow PEP 8 guidelines for Python code.
- Write clean, understandable code without unnecessary comments (as per project standards).
- Test your changes locally before submitting a PR.
